NOSSA AMBIÇÃO

Proporcionar aos nossos clientes uma jornada segura de migração para a nuvem resultando em performances à altura dos seus negócios

SITUAÇÃO DE HOJE

  1. Inúmeras migrações para a nuvem são feitas:
    • de maneira apressada
    • sem o planejamento adequado
    • com deficiência no estabelecimento de requisitos
    • sem a campanha de testes necessária para grarantir a transição sem problemas
    • com ausência de um plano de contigência para enfrentar eventos inesperados

Portfólio de Serviços

  1. Levantamento da situação do cliente;
    • com vistas à migração para a nuvem, ou
    • com processo de migração problemático
  2. Desenvolvimento de estratégia para migração
  3. Desenvolvimento de soluções para problemas de migração
  4. Elaboração de RFP(Request for Proposal)
    • Request for Information(RFI)
    • RFQ "Request for a Quotation"
    • com processo de migração problemático
  5. Apoio no processo de seleção de contratação de empresas para implantação
  6. Apoio na fiscalização da migração para a nuvem
  7. Monitoramento da operação de negócio pós migração para a nuvem
  8. Elaboração de planos de contigência e continuidade de negócios
  9. Agregar talentos/competências críticas para compor as equipes de projeto dos clientes
